# Topic-in-Business-Analytics

Code and project files for Exeter Cycle Hire feasibility analysis (BEMM457).


https://drive.google.com/file/d/1ErD8pxG1k908nX70gOnxckSMdojlmBOf/view?usp=sharing - pbix file
How to view interactively:
Open the link and download the .pbix file.
Open it in Power BI Desktop.
Use the slicers (e.g., Weekday/Weekend) and visual cross-filtering to explore demand patterns, top stations, and trip duration distributions.
